# Mega-Sena

Flutter application that presents a concept of the brazilian lottery (Mega-sena) [website](http://loterias.caixa.gov.br/wps/portal/loterias/landing/megasena/) using mobx to manage states and provider to inject dependencies.

### Tools
- [Mobx](https://pub.dev/packages/mobx)
- [Provider](https://pub.dev/packages/provider)
- [Google fonts](https://pub.dev/packages/google_fonts)
- [Intl](https://pub.dev/packages/intl)
